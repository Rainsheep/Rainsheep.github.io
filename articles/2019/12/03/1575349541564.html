PAT排名汇总

2019-12-03
acm, STL, 模拟 0 评论 0 浏览

输入格式：

输入的第一行给出一个正整数 N（\le≤100），代表考点总数。随后给出 N 个考点的成绩，格式为：首先一行给出正整数 K（\le≤300），代表该考点的考生总数；随后 K 行，每行给出 1 个考生的信息，包括考号（由 13 位整数字组成）和得分（为[0,100]区间内的整数），中间用空格分隔。

输出格式：

首先在第一行里输出考生总数。随后输出汇总的排名表，每个考生的信息占一行，顺序为：考号、最终排名、考点编号、在该考点的排名。其中考点按输入给出的顺序从 1 到 N 编号。考生的输出须按最终排名的非递减顺序输出，获得相同分数的考生应有相同名次，并按考号的递增顺序输出。

输入样例：

 12
 25
 31234567890001 95
 41234567890005 100
 51234567890003 95
 61234567890002 77
 71234567890004 85
 84
 91234567890013 65
101234567890011 25
111234567890014 100
121234567890012 85

31234567890014 1 2 1
41234567890001 3 1 2

学号不能 Longlong，存入 vectore 的数据是个副本，修改里面的数据无法传到外边，即原来数据不会被修改，且 vectore 不能引用(vectore<int&>)，只能传入指针，修改原数据。注意调用数据的方式，(*it)->score，必须加括号，-> 优先级大于*

 1#include<iostream>
 2#include<algorithm>
 3#include<vector>
 4using namespace std;
 5struct node
 6{
 7	string num;//不能用Longlong，longlong第一位为0会自动删除 
 8	int score;
 9	int sumrank;
10	int rank;
11	int work;
12}student[30005];//存学生信息 
13bool cmp(node* a,node* b)
14{
15	if(a->score!=b->score)
16	return a->score>b->score;
17	return a->num<b->num;
18}//比较函数，分数不等按学号排 
19int main()
20{
21	int T,t,n,i,j,k,s=0;
22	vector<node*> q;//vector存指向结构体的指针 
23	vector<node*>::iterator it;
24	cin>>T;
25	for(i=1;i<=T;i++)
26	{
27		q.clear();//清空，否则会在原基础上存数据 
28		cin>>n;
29		while(n--)
30		{
31			cin>>student[s].num>>student[s].score;
32			student[s].work=i;
33			q.push_back(&student[s]);//压入结构体的指针 
34			s++;//学生数量加1 
35		}
36		sort(q.begin(),q.end(),cmp);
37		(*q.begin())->rank=1;//注意这地方的运用，(*q.begin())是指向结构体的指针，必须加括号，->的优先级大于*，*a.b等级于a->b 
38		for(it=q.begin()+1,j=2;it!=q.end();it++,j++)
39		{
40			(*it)->rank=j;//(*it)是结构体的地址 
41			if((*it)->score==(*(it-1))->score)(*it)->rank=(*(it-1))->rank;
42		}	
43	}
44   q.clear();
45   for(i=0;i<s;i++)
46   q.push_back(&student[i]);
47	
48	
49		sort(q.begin(),q.end(),cmp);
50	(*q.begin())->sumrank=1;
51	for(it=q.begin()+1,j=2;it!=q.end();it++,j++)
52	{
53		(*it)->sumrank=j;
54		if((*it)->score==(*(it-1))->score) (*it)->sumrank=(*(it-1))->sumrank;
55	}
56	
57	cout<<s<<endl;
58	for(i=0;i<s;i++)
59	cout<<q[i]->num<<" "<<q[i]->sumrank<<" "<<q[i]->work<<" "<<q[i]->rank<<endl;
60	
61}
