录屏软件Bandicam设置教程

2019-12-17
软件教程 0 评论 0 浏览
参考链接 bandicam软件基础入门

鼠标设置

  • 点击效果：左键右键有圈圈
  • 高亮效果：鼠标高亮，没啥用

音频设置

  1. 单独保存音频文件
  2. 主音频为系统声--保持默认或者选择正在使用的扬声器设备。（系统声音）
  3. 副音频为麦克风——保持默认或者选择正在使用的麦克风设备。（麦克风声音）
  4. 混录主副音频设备所再现的声音是将系统音频和麦克风的音轨混合成一个音轨，勾选后就无法在音频编辑软件上分别编辑了。

选项

  • 在选项中有【对于游戏录制模式】这一项，有个当帧率不更新时跳过录制。主要影响：在游戏读取或者载入时一般帧率就会不更新保持一个定值，此时如果勾选了这个选项，bandicam 会跳过该段的录制。（建议取消勾选）

录像设置

  • 软件提供了几个预设，可以直接选用。

此处重点讲自定义配置

  1. 主流格式 mp4，mp4 格式支持率高，视频体积小，视频质量好。
  2. 尺寸根据自己需求选用。
  3. FPS 使用 30fps，是指画面每秒传输帧数（各大视频网站不支持 30fps 以上的帧数，国外 YouTube 支持 60fps）。
    3.1 fps 右边的省略号
    • VFR 和 CFR 是动态帧数和恒定帧数。选用 VFR 来达到最佳效果。
      注：CFR 保证 FPS 的稳定性让视频的帧数不会突然降低造成视觉上的卡顿。但是对于机器性能不足电脑配置差的用户，该选项就失去了作用。若要用 PR 剪辑，则需要使用恒定频率。
  4. Codec 编码模式：

    H264(CPU)：使用硬件 CPU 加速。
    H264(NVIDIA)：使用显卡加速。
    MPEG-4：一般不选。
    CPU 性能好的使用 H264（CPU 加速），显卡性能好的使用 H264（显卡加速），两者都没有或者都不能用的选用 MPEG-4。

    码率控制 VBR 和 CBR。 VBR 是动态码率，会动态调节码率保证画面品质，比如在游戏内黑暗的环境中 CBR 会出现灰色方格，以及在高速变换画面的 FPS 射击游戏中保证画面不会模糊。而 CBR 固定码率是保证码率的固定，缺点明显，会造成画面的问题。
    优化方面基本使用更快的编码速度来减少对游戏的影响。
    关键帧间隔：120~150
